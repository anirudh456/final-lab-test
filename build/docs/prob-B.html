1 Group B Problem set

You are given the implementation of the show_users route. Please study the implementation and the test cases to understand how the route is implemented across the two components: guard and engine.

Now, your job is to complete the implementation of the guard and engine components for del-user user route.

Follow these steps:

1.1 Guard

  1. Go to the del_user directory that contains a command handler DelUser.do.
  2. Go to the Checks section of the del_user page of Guard. Under each of its subsections, you will find incomplete definitions of methods, namely:
    • check_type
    • check_auth
    • check_state
    • Complete the definitions of the above methods.
  1. Build and test. There are some test cases in the

del_user tests. Please study them and make sure that they continue to PASS!

  1. Write the test case test_check_auth. Complete the

TestCheckAuth test case by providing a definition of the test_check_auth method. This test is supposed to raise an exception because the session's role is not admin.

1.2 Engine

  1. Go to the del_user directory of the engine that contains a skeleton of the command handler DelUser.do.
  1. Complete the do method by invoking the appropriate

method on the entity manager.

  1. Make sure test cases continue to PASS!
