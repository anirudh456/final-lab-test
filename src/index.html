1 First things first

  1. Questions are allowed only about the initial build.
  2. No questions will be entertained after the 10am. Please proceed after making reasonable assumptions after that.

  3. Literate code is under ./runtime. Built code is under ../build/code and documentation under ../build/docs.

    You are expected to conform strictly to the above structure. If you need to create any new directories, please make sure they are in the tree under ./runtime. If you fail to do so (e.g., create a new directory under src), you may incur major penalties.

  1. Please sit ONLY at the seat assigned to you by your TA's.
  1. Fill in the org file ./student-info.html. Do this now Please clearly mention your seat number.
